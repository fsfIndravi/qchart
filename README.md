# QChart

Легковесная библиотека для построения графиков котировок для мобильных устройств.

## Особенности
- Быстродействие, прорисовка только видимой части графика
- Отсутствие зависимостей
- Компактный код
- Гибкие настройки

## Установка
```
npm i qchart --save-dev
```
